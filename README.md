# Sorting-Visualizer
A simple and interactive web-based Sorting Visualizer built using plain HTML, CSS, and JavaScript. This tool helps visualize different sorting algorithms step-by-step, making it easy to understand how they work.

**Sorting Algorithms**: 
Visualizes common sorting algorithms like:
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort

**Project Structure**
index.html: Contains the structure of the webpage.
style.css: Handles the visual styles of the project.
script.js: Includes the logic for the sorting algorithms and the visualization.

**Technologies Used**
HTML: For structuring the webpage.
CSS: For styling the visual elements.
JavaScript: For implementing the sorting algorithms and handling the DOM manipulation for the visualizer.
